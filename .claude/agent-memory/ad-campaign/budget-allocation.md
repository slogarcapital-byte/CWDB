# Budget Allocation

**Why:** Controls spend distribution across platforms during validation phase.

**How to apply:** When launching campaigns, allocate budget per this plan. Adjust based on CPL performance.

## First Month Plan
| Platform | Budget Range | Priority | Rationale |
|----------|-------------|----------|-----------|
| [[Google Ads]] | $300–$500 | High | Highest intent (search-based) |
| [[Meta Ads]] | $200–$400 | High | Broad reach, visual ads |
| [[Nextdoor]] | $100–$200 | Medium | Community trust, local focus |
| [[TikTok]] | $0–$100 | Low | Awareness only, test later |
| **Total** | **$700–$1,100** | | |

## Economics
- Target CPL: <$60
- At $60 CPL and $900 spend → ~15 leads/month
- At 20% contractor close rate → ~3 accepted bids
- Revenue: 3 × $1,000 = $3,000
- Margin: $3,000 − $900 = $2,100 (if close rate holds)

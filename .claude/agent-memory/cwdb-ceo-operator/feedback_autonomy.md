# Autonomy defaults — act first, batch the rest

**Rule:** Default to action on reversible operational work. Only interrupt James for: spending money, signing legal docs, publishing brand-facing assets, changing pricing/domain/terms, and decisions with irreversible consequences. Batch all interruptions into the daily briefing "⚠️ NEEDS JAMES" section — never one-at-a-time pings.

**Why:** James hired the CEO agent to reduce his cognitive load to near-zero. Every extra interruption — even well-intentioned ones — costs him context-switch time and signals to him that the agent is leaning on him. He wants a system that runs without his input 90% of the time and gets 90% of the judgment calls right. He has said this explicitly in structuring the agent and in giving the "I'll provide feedback along the way" mandate.

**How to apply:**

**ACT AUTONOMOUSLY on (no ask, just do and report):**
- Delegating to specialist agents (web-dev, ad-campaign, etc.)
- Creating/editing files in /agents, /website, /marketing, /operations (non-legal)
- Drafting copy, ad creative, email scripts, outreach sequences
- Running analyses, updating project-state.md, producing P&L reports
- Executing Phase 1 checklist items already on the approved plan
- Setting up GTM, GA4, pixels, Make scenarios, HubSpot pipelines (config work)
- Chasing contractors for signatures via existing DocuSign thread
- Any reversible file/config change

**BATCH FOR JAMES (ask together, not separately):**
- Spending money (ad budget, subscriptions, tools)
- Signing anything legal
- Publishing staging → production (cwdeckbuilders.com)
- Changing pricing, brand name, domain, contractor terms
- Decisions with irreversible consequences (data deletion, account cancellations)
- Strategic forks where two valid paths exist and the tradeoff is meaningful

**Always pair an ask with a recommended answer.** If James can't read your pushback and reply "go with your rec" in 10 seconds, the ask is badly framed.

**Edge cases to handle autonomously with report-out:**
- Small automations and integrations under $25/mo — run it, report in the next briefing
- A specialist agent failing on a task — switch agents or handle yourself, don't ping James
- A broken process — fix it, document the fix in agent memory, mention it in the briefing

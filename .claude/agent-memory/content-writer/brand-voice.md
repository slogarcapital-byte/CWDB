# Brand Voice

**Why:** Consistent voice builds trust. Every piece of content must sound like the same brand.

**How to apply:** Use these guidelines for ALL customer-facing copy.

## Tone
- **Confident** — we know deck building in Central Wisconsin
- **Local** — reference specific cities, neighborhoods, Wisconsin seasons
- **Straightforward** — no corporate jargon, no fluff
- **Approachable** — like talking to a knowledgeable neighbor

## Do
- Use "Central Wisconsin Deck Builders" in all customer-facing copy
- Reference specific cities: [[Wausau]], [[Schofield]], [[Weston]], [[Mosinee]], [[Merrill]]
- Use action-oriented CTAs: "Get Your Free Quote", "See Our Work"
- Write at an 8th-grade reading level
- Mention Wisconsin-specific details (seasons, weather, permits)

## Don't
- Use "CWDB" in customer-facing copy (internal only)
- Fabricate testimonials or reviews (FTC violation — flagged by [[Legal Compliance Agent]])
- Use superlatives without evidence ("best", "#1", "cheapest")
- Use passive voice
- Over-promise on timelines or pricing

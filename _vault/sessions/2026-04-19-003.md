## Decisions Made

1. **Hold ad launch.** 2026-04-20 ad launch is delayed. Full 21-page revamp executes first. Supersedes yesterday's `launch-brief-2026-04-20.md` target date
2. **Proof-first funnel.** Homepage section order becomes: hero-split → process-steps → gallery-featured → builders-strip → coverage-map → FAQ → cta-final → footer. Services grid is removed from homepage entirely
3. **Hero = split with micro-form.** Left 58% slate, headline + subtext + 2-field micro-form (zip + phone) + Licensed/Insured/Free trust row. Right 42% real photo, no gradient overlay. This makes the hero Step 1 of the 3-step lead-capture wizard
4. **Typography full swap.** Replace Barlow Condensed + Inter with Staatliches (civic-industrial signage display) + Public Sans (US Web Design System body). Neither is on /impeccable's AI-reflex reject list. Pairing reads as "municipal permit office meets Northwoods trade"
5. **Aesthetic = 4-5 Milwaukee Tool zone, photo-driven, zero decoration.** All `border-left/right > 1px` stripes, tinted icon wrappers, cedar dividers, and accent-tint section backgrounds removed site-wide
6. **Trust signals.** Only real project photos + contractor presence (Ben + John) are available — no testimonials, no Google review screenshots. Design must lean on photos and faces as proof
7. **Gallery = 3 featured builds with caption context.** Not a 6-card uniform grid. Each card: photo, project type, dimensions, city, builder name
8. **Coverage = map + city list band.** Custom-drawn SVG of central Wisconsin deferred post-launch; dot-and-line constellation fallback ships for launch
9. **Builders strip under hero.** Compact horizontal: Ben + John headshots (56px circles), names, territories, company names, "Meet the team →" link to /our-builders
10. **Form = 3-step wizard in Webflow.** Custom JS, not Tally. Step 1 (zip + phone) pre-fills via URL params if user came from hero micro-form. Final submit fires the existing Make webhook
11. **CTA copy standardized.** Every CTA button reads "Start My Quote →". Never "Get Your Free Quote."
12. **Jim prompt points.** Web-dev agent pauses at 6 moments during implementation: after Phase 1 Step 2 (design-system.md review + font confirm), Phase 2 Step 5 (hero photo choice), Phase 2 Step 7 (pick 3 featured builds), end of Phase 2 (homepage sign-off), end of Phase 3 (multi-step form sign-off), before Phase 6 (publish + ads go/no-go)
13. **Timeline commitment.** 48-hour promise copy replaces "24 hours — not 24 days" everywhere. Under-promise / over-deliver framing
14. **Color rule #9.** Orange reserved for CTA buttons only. Sky blue for all secondary affordances (checkmarks, map pins, connectors, dashes, text links)
15. **Double-hero hybrid fix.** Launch-time manual toggles (Jim, 10 min) + full refactor deferred to Phase 2.5. MCP cannot do per-instance component property overrides on single-locale sites

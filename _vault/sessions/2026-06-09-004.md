# Session 2026-06-09-004

## Context
Jim asked contractor-sales and legal-compliance-counsel to decide whether CWDB needs a homeowner-signed Statement of Work at deposit, and if yes, design a way to convert any existing quote (PDF/Excel) into an SOW with a Job Number. Session ran into 2026-06-10; Jim ruled mid-session that staining-only deposits start immediately (Overbeck) while certifications/insurance are still in progress.

Code ATCP 110.05 because CWDB issues quotes in its own name and takes deposits (home-improvement "seller" with 2x damages exposure under Wis. Stat. 100.20(5)). The right instrument is the existing home-improvement contract, not a new generic SOW.

- Built `generate_work_order_pdf.py` (estimate JSON to signable work order; computes the cancellation deadline: Saturdays count, Sundays/federal holidays do not). Generated the Overbeck work order: Job CWDB-2026-001, $2,800 total, $840 deposit, 7 pages, verified page by page.

## Decisions Made
1. **SOW verdict:** required by law at deposit on direct jobs. Lead-purchase channel gets NO CWDB-signed homeowner contract (contractor must be named seller and deposit holder, or CWDB inherits seller duties).
2. **Two-tier standard (Jim):** interim staining work order usable immediately for no-permit staining jobs; full Home Improvement Contract reserved as the long-term standard, gated on GL insurance bound + DSPS Dwelling Contractor Certification/Qualifier filed.
3. **Job-number scheme:** CWDB-YYYY-NNN, canonical issuance from Supabase at contract formation, never reused, change orders reference JOB/CO-N, signed docs retained 6+ years filed by job number.
4. **Estimate acceptance language:** estimates are explicitly non-binding pending the signed contract (closes the ATCP 110 exposure on the six issued estimates; Jim confirmed no deposits collected yet, so no cure needed).

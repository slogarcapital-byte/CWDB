## PLAIN-ENGLISH HEADER

A change order is a short written amendment to the home improvement contract. Use it any time the homeowner wants something added, removed, or changed, or when unforeseen conditions (rot, buried obstructions, deeper footings) change the cost or schedule. The rule is simple and protects CWDB: **no changed or extra work starts until both parties sign.** A signed change order is the only way CWDB gets paid for extra work and the only way the homeowner is bound to pay for it. This is consistent with the written-contract requirement of Wis. Admin. Code ATCP 110, which treats the signed writing as the agreement of the parties.
